# Portfolio – Loïc Rakotondrasoa

Petit portfolio personnel de graphiste / webdesigner, réalisé avec **React**, **Vite** et **Tailwind CSS**.

## Installation

```bash
cd app
npm install
```

## Lancer le projet en local

```bash
cd app
npm run dev
```

Puis ouvrir l’URL indiquée dans le terminal (généralement `http://localhost:5173`).

## Build de production

```bash
cd app
npm run build
```

Les fichiers de production seront générés dans le dossier `app/dist`.
